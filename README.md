# Bento

A cute pixel-art robot head that lives in your browser. Tap him, pet him, or just let him do his thing — he chirps, blinks, gets curious, takes naps, and reacts to your taps.

Built with Vite, Canvas API, and Web Audio API. Zero dependencies.

## Quick Start

```bash
cd bento
npm install
npm run dev
```

Open `http://localhost:5173/bento/` and tap Bento.

## Deploy

Push to `main` and GitHub Actions auto-deploys to GitHub Pages.

```bash
git push origin main
```

## Features

- Animated pixel-art eyes on a visor display
- Synthesized chirps, boops, and sounds (Web Audio API)
- Reacts to taps with happy bounces
- Random events: glitches, spins, sneezes, rainbow visor, heart eyes
- Emotions: curious, excited, confused
- Takes naps (zzz floating above head)
- A friendly firefly visits sometimes
- Skin system for theming

## Built With

- [Vite](https://vitejs.dev/) — Build tool
- Canvas API — Pixel-art rendering
- Web Audio API — Sound synthesis
